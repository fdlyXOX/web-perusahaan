Cloud Architect

Deskripsi Pekerjaan

Sebagai Cloud Architect di PT. NETSOLUSI TEKNOLOGI, Anda akan bertanggung jawab untuk merancang dan mengimplementasikan solusi cloud yang scalable dan aman untuk klien kami. Anda akan bekerja sama dengan tim IT dan pengembang untuk memastikan arsitektur cloud memenuhi kebutuhan bisnis dan teknis.

Posisi ini memerlukan pemahaman mendalam tentang layanan cloud, keamanan, dan best practices dalam desain arsitektur cloud.

Tanggung Jawab Utama

  • Mendesain arsitektur cloud yang scalable dan aman
  • Bekerja dengan tim pengembang untuk mengimplementasikan solusi cloud
  • Mengelola migrasi aplikasi dan data ke cloud
  • Memastikan kepatuhan terhadap standar keamanan dan regulasi
  • Memonitor performa dan biaya penggunaan cloud
  • Mengembangkan dokumentasi arsitektur dan prosedur operasional
  • Memberikan pelatihan dan dukungan teknis terkait cloud

Kualifikasi dan Persyaratan

Pendidikan:

  • Sarjana Teknik Informatika, Sistem Informasi, atau bidang terkait
  • Sertifikasi cloud seperti AWS Certified Solutions Architect, Microsoft Azure Architect adalah nilai tambah

Pengalaman:

  • Minimal 3 tahun pengalaman dalam arsitektur cloud dan implementasi
  • Pengalaman dengan platform cloud utama seperti AWS, Azure, atau GCP
  • Pengalaman dalam migrasi cloud dan manajemen infrastruktur cloud

Keahlian Teknis:

  • Pengetahuan mendalam tentang layanan cloud dan arsitektur microservices
  • Pengalaman dengan containerization (Docker, Kubernetes)
  • Keahlian dalam scripting dan automation (Python, Terraform, Ansible)
  • Pengalaman dengan keamanan cloud dan compliance

Keahlian Soft Skills:

  • Kemampuan analisis dan pemecahan masalah yang kuat
  • Kemampuan komunikasi yang baik untuk berkolaborasi dengan tim dan klien
  • Kemampuan bekerja dalam tim dan berkolaborasi lintas departemen
  • Komitmen terhadap pembelajaran berkelanjutan di bidang cloud computing

Keuntungan Bergabung

  • Gaji kompetitif sesuai pengalaman
  • Tunjangan kesehatan dan asuransi
  • Pelatihan dan sertifikasi cloud berkelanjutan
  • Proyek-proyek menantang dengan teknologi terkini
  • Tim profesional dan lingkungan kerja yang mendukung
  • Peluang karir dan pengembangan profesional
  • Work-life balance yang baik
